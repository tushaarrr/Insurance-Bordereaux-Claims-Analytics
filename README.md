# Insurance Bordereaux & Claims Analytics Dashboard

## Overview

This project is an Excel-based insurance analytics solution designed for P&C Insurance / Actuarial Data Analyst roles. It converts raw policy and claims data into business-ready insights through data cleaning, KPI reporting, advanced formulas, and dashboard visualization.

The goal was to simulate a real-world bordereaux reporting workflow used in insurance operations, pricing, reserving, and portfolio monitoring.

## Tools Used

* Microsoft Excel
* Advanced Excel Formulas
* INDEX/MATCH
* VLOOKUP / XLOOKUP
* SUMIFS / COUNTIFS
* Charts & Dashboarding
* SQL (for analytical querying)

## Project Files

* **Insurance_Bordereaux_Analytics_Dashboard.xlsx**
  Main workbook containing dashboard, cleaned data, KPI sheets, and formula analysis.

* **Insurance_SQL_Queries.txt**
  SQL table creation script and analytical queries.

## Workbook Structure

### Raw_Data

Original imported source data.

### Cleaned_Data

Prepared dataset with derived business metrics and reporting fields.

### KPI_Data

Summary metrics used in dashboard reporting.

### Lookup_Analysis

Examples of lookup formulas and policy-level analysis.

### Dashboard

Executive view of portfolio performance with charts and KPIs.
